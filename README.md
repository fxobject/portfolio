Система биллинга для оператора связи
Разработка проводилась на (с использованием) :
 node.js, express, handlebars, vue, mssql, MongoDB, typescript, WEBPACK, SASS
 взаимодействие клиент - сервер реализовано по протоколам http,websocket
 под специфические требования для node были реализованы отдельные сетевые протоколы (dhcp)
